Vim: Extracting a Block to a New File

1. Open file:
   vim yourfile.tex

2. Move to block start
3. Press V to start visual line mode
4. Move to block end

5. Yank or cut:
   y   (copy)
   d   (cut)

6. Open new file:
   :split newfile.tex

7. Paste:
   p

8. Save and close:
   :wq

9. Return to original:
   :bp   or   Ctrl-w w

10. (LaTeX optional):
    \include{newfile}

11. Undo / Redo:
    u     (undo)
    Ctrl-r (redo)

12. Jump between files:
    :n     (next)
    :prev  (previous)

13. Search within file:
    /pattern     (forward)
    ?pattern     (backward)
    n / N        (repeat forward/backward)
